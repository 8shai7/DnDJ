# DnDJ by Bass Union

**Professional audio workflow for DJs and producers** — pull tracks from YouTube, shape them in one place, tag them, and get them ready for your sets or USB export.

---

## Why DnDJ?

| You get | Details |

| **Faster prep** | Search, download, and analyze without juggling five browser tabs and separate editors. |
| **Built-in polish** | Trim, fades, and waveform editing with BPM and musical key detection. |
| **Organized output** | ID3 tagging so your library and decks stay readable. |
| **Licensed for real use** | Per-machine activation via **Lemon Squeezy** — buy once, activate online, keep working offline after activation. |

DnDJ is a **desktop app** (Windows-focused builds; macOS support where provided). Your license is tied to your machine for fair use — no subscription games unless you choose a subscription product on the store.

---

## Features

- **Discover & download** — Find versions on YouTube and download audio (e.g. MP3 / WAV / FLAC) with quality options.
- **Waveform & trim** — Visual editing: set in/out, fades, and preview playback.
- **BPM & key** — Automatic analysis to speed up harmonic mixing and set prep.
- **Metadata** — Edit and embed **ID3** tags so titles, artists, and artwork show up correctly.
- **USB / export workflows** — Prepare folders and tracks for sticks and gigs (where enabled in your build).
- **Updates** — **yt-dlp** is refreshed in the background so extractors stay current between app releases.
